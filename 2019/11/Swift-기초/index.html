  • 한 라인에 여러구문을 사용하고 싶을 때 사용(다중명령)

var vs let

  • 상수: 값 변경이 불가능 -> var
  • 변수: 설정한 값을 추후에 변경가능 -> let

변수네이밍

영어 외에도 유니코드 문자를 포함한 대부분의 문자를 사용해 네이밍 가능

사용할 수 없는 네이밍

변수로 사용할 수 없는 이름

  1. 같은 스코프 범위 내에서 중복되는 이름
  2. 공백문자 let bla nk = "123"
  3. 수학 기호 (√, ∑ 등)
  4. 화살표 (→, ← 등)
  5. 숫자로 시작하는 이름 (시작 부분 외에는 사용 가능) let 123변수 = "456"

Literals & Type

  • 리터럴
  • 소스코드에서 고정된 값으로 표현되는 문자 (데이터) 그 자체
  • 정수 / 실수 / 문자 / 문자열 / 불리언 리터럴 등

Integer

  • 8-bit : Int8, UInt8
  • 16-bit : Int16, UInt16
  • 32-bit : Int32, UInt32
  • 64-bit : Int64, UInt64
  • Platform dependent(플랫폼 의존성을 가진 Int) : Int, UInt

현대의 기기는 64bit

Floating-point Literal

Swift에서 소수점은 기본적으로 Double로 받아들인다.

Double 8Byte
Float 4Byte

Double - 최소 소수점 이하 15 자리수 이하의 정밀도

Float - 최소 소수점 이하 6 자리수 이하의 정밀도

부동 소수점이므로 소수점 이하의 정밀도는 변경 될 수 있음
