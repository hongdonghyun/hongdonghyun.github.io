문제

하샤드 수 구하기

  • 하샤드 수 : 자연수 N의 각 자릿수 숫자의 합을 구한 뒤, 그 합한 숫자로 자기 자신이 나누어 떨어지는 수

e.g. 18의 자릿수 합은 1 + 8 = 9 이고, 18은 9로 나누어 떨어지므로 하샤드 수.

Code

풀이

입력받은 숫자의 모든자리 합
해당 내용을 응용하면 금방 구현이 가능하다.

하샤드 수를 구하기 위해선 자릿수를 모두 더해줘야 하기 때문에
나온 결과에서 자기자신으로 나누어 떨어지는지 확인하면 된다.
