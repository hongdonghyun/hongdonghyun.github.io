Collection

종류

Value Type (Swift Collection)

  • Array
  • Dictionary
  • Set

Object Type (Foundation Collection)

  • NSArray, NSMutableArray
  • NSDictionary,NSMutableDictionary
  • NSSet, NSMutableSet

Array

  • 값이 중복될 수 있으며 순서가 보장된다.
  • 순서의 시작은 0

선언방법

타입추론

타입선언

method

.isEmpty : 값이 비어있는지 체크 -> Bool

.startIndex : 시작 인덱스 값 -> Int

.endIndex : 마지막 인덱스 값 -> Int

.contains(value) : value가 들어있는지 체크 -> Bool

.firstIndex(of: value) : 해당 value가 들어있는 첫번째 위치 -> Optional

insert

.insert(추가하고 싶은 값, at: 추가하고 싶은 위치)

operator

배열끼리 연산자 적용이 가능하다.

Set

값이 중복될 수 없으며 순서가 보장되지 않는다.

Dictionary

  • Key : Value 쌍으로 이루어져 있다.
  • 순서가 보장되지 않는다.
  • Key는 중복 될 수 없다.

타입선언

method

.keys : 딕셔너리의 key값만 출력 -> Array

.values : 딕셔너리의 value값만 출력 -> Array

.contains(where: { (key, value) -> Bool in return key == "A"}) : key 또는 값 찾기

Set

  • 순서를 보장하지 않는다.
  • 유일한 값을 지님

선언방법

method

.hashPrefix(value) : 해당 값으로 시작하는지 체크 -> Bool

.contains(value): 해당 값이 요소로 존재하는지 체크 -> Bool

값 추가

set.insert(value) -> (inserted: Bool, memberAfterInsert: String)
