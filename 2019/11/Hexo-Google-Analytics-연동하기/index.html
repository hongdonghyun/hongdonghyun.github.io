구글 애널리틱스 연동하기

Hexo에 방문자 통계, 페이지 조회를 확인할 수 있는게 없는것 같다.

그래서 구글 애널리틱스를 달아서 확인해보도록 하자.

Google Analytics 가입

여기로 가면 구글 로그인을 요구하게 된다.

해킹아닙니다.

로그인을 하고나면 구글 애널리틱스 메인화면으로 오게 된다.

설정

  1. 무료로 계정 만들기 버튼 클릭
  1. 계정이름 설정
  1. 웹페이지에 연동할것이므로 웹을 선택
  1. 속성 설정
  1. 약관동의

스샷 없음

잠깐

만약 애널리틱스 계정이 있거나 한번이라도 보고서를 만든적이 있다면 해당 화면처럼 진행이 안됩니다.

그런경우 좌측하단 관리버튼 -> 속성 만들기를 통해 진행해주세요.

추적 ID 확인

빨간색 테두리 부분의 추적 아이디를 확인하세요.

적용

추적 ID를 확인한 후, 각자 블로그의 테마폴더로 이동하자.

여기

_config.yml파일을 열어 700번째 줄 근처에 구글 애널리틱스 관련 설정이 있다.

저기 tracking_id부분에 아까 발급받았던 아이디를 입력해주면 된다.

ex)

적용을 했다면 블로그 최상단 디렉토리로 이동후 hexo g -d를 입력하면 적용이 완료된다.

hexo g -d 명령어는
generate && deploy 입니다.
