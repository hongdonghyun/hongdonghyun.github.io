[Swift]정수의 합

작성일 2019-11-21 Edited on 2019-11-25 In Algorithm Views: Disqus:

문제

  • 정수 하나를 입력받은 뒤, 해당 숫자와 숫자 1사이에 있는 모든 정수의 합계 구하기
  • e.g. 5 -> 1 + 2 + 3 + 4 + 5 = 15
  • -2 -> -2 + -1 + 0 + 1 = -2

풀이

입력값을 절대값으로 바꿔 계산 후
음수의 경우와 양수의 경우로 나누어 리턴
0일 경우에도 따로 리턴
