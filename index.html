작성일 2020-01-10 In Algorithm Disqus:

문제

프로그래머스 링크

Code

풀이

Integer값이 주어진걸 자릿수를 더하는 문제다.

처음엔 forEach를 사용하여 풀려고 했지만

고차함수를 사용하면 더 간결한 코드가 나오지 않을까 고민을 해보았다.

고차함수를 많이 써본적이 없어서 연습할겸 시도해보았다.

Integer를 문자열로 바꾸고 map함수를 사용해 한자리씩 정수로 다시 바꾸고 그 결과를 reduce를 통해 더해주었다.

String의 값을 하나씩 빼오면 Character타입이 되는데 이상태에서 바로 정수형으로 바꾸어지지가 않아서 String -> Int로 이중 형변환을 사용하여 Integer로 바꾸어 주었다.

고차함수를 쓰니 많이 간결해졌다.

축약하는 순서에대해서도 공부해보아야겠다.

[Swift] 이상한 문자 만들기

작성일 2020-01-08 In Algorithm Disqus:

문제

프로그래머스 링크

code

풀이

Character에 메서드를 추가해서 풀어보았다.

맨처음 생각난 풀이는 문자열을 공백기준으로 split하여 2중for문으로 푸는 방법이 생각났는데,

죽어도 2중for문은 쓰기 싫어서 고민을 한번해보았다.

extension으로 isAlpha라는 메서드를 만들어 해당 문자가 영어인지 판별 후

영어인 경우 문제의 조건에 따라 대문자 소문자로 변경해 결과값 문자열에 추가하고,

영어가 아닌경우를 공백으로 판단하여

사실 이경우 특수기호,한글등이 하나라도 들어가 있다면 난장판이 되겠지만
문제의 조건이 알파벳을 대문자,소문자로 구분을 해야하기 때문에
알파벳, 공백이외의 문자는 들어오지 않을거라는 판단하에 진행하였다.

공백을 결과값 문자열에 추가해주었다.

extension에 익숙하지 않았는데 생각보다 재미있게 풀어서 좋았다.
