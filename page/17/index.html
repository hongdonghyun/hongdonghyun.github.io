구글 애널리틱스 연동하기

Hexo에 방문자 통계, 페이지 조회를 확인할 수 있는게 없는것 같다.

그래서 구글 애널리틱스를 달아서 확인해보도록 하자.

Google Analytics 가입

여기로 가면 구글 로그인을 요구하게 된다.

해킹아닙니다.

잠깐

그런경우 좌측하단 관리버튼 -> 속성 만들기를 통해 진행해주세요.

과제풀이

학점을 입력받아 각각의 등급을 반환해주는 함수

특정 달을 숫자로 입력 받아서 문자열로 반환하는 함수

세 수를 입력받아 세 수의 곱이 양수이면 true, 그렇지 않으면 false 를 반환하는 함수 (switch where clause 를 이용해 풀어볼 수 있으면 해보기)

반복문(for , while , repeat - while)을 이용해 아래 문제들을 구현해보세요.

자연수 하나를 입력받아 그 수의 Factorial 을 구하는 함수

자연수 두 개를 입력받아 첫 번째 수를 두 번째 수만큼 제곱하여 반환하는 함수

자연수 하나를 입력받아 각 자리수 숫자들의 합을 반환해주는 함수

자연수 하나를 입력받아 1부터 해당 숫자 사이의 모든 숫자의 합을 구해 반환하는 함수를 만들되, 그 합이 2000 을 넘는 순간 더하기를 멈추고 바로 반환하는 함수

1 ~ 50 사이의 숫자 중에서 20 ~ 30 사이의 숫자만 제외하고 그 나머지를 모두 더해 출력하는 함수

문자열 두 개를 입력받아 두 문자열이 같은지 여부를 판단해주는 함수

자연수를 입력받아 그 수의 약수를 모두 출력하는 함수

100 이하의 자연수 중 3과 5의 공배수를 모두 출력하는 함수

두 개의 자연수를 입력받아 두 수를 하나의 숫자로 이어서 합친 결과를 정수로 반환하는 함수

2 이상의 자연수를 입력받아, 소수인지 아닌지를 판별하는 함수

헥소 댓글창 만들기

본 포스팅은 next 테마를 기준으로 설명합니다.

Disqus

Disqus는 소셜 댓글 서비스

댓글영역과 로직을 직접 구현하지 않아도 위젯 형태로 서비스를 제공함

특별한 기능을 사용하지 않는다면 basic플랜(공짜)를 사용하면 된다.

라이브리라는 소셜 댓글 서비스가 있지만(국산), next 테마에서 특별히 지원하지 않기 때문에 disqus로 간단하게 댓글을 달아보도록 한다.

Disqus 가입

https://disqus.com/로 이동하고 우측 상단에 Get Started를 누르면 회원가입을 하면 된다.

너무쉬워서 스샷도 없음

생성

https://disqus.com/다시 메인페이지로 와서 우측 상단에 로그인이 된것을 확인하고
화면 가운데 GET STARTED를 누른다.

화면 가운데 GET STARTED 버튼을 누르면

다음과 같이 뜬다.

이제 빨간색 테두리영역이 있는 부분을 눌러준다.

그럼 다음과 같이 정보를 입력하는 창이 뜨는데
정보를 맞춰 입력 해주면된다.

Website Name : 사이트이름 (디스커스에서 관리할때 보인다.)

Category : 해당 사이트 게시물이 올라오는 타입

저는 Tech

설정을 다하고 넘어가면

플랜을 설정하라고 하는데 밑으로 내리면 무료플랜이 있다.

Subscirbe Now버튼 클릭

무료플랜이 밑에있어서 당황스러웠습니다.

그러면 디스커스를 적용할 사이트들 목록이 나오는데 여기에 Hexo가 없습니다.

당황하지 마시고 밑으로 쭉 내리시면 Universal Code가 있습니다.

해당 버튼 클릭 -> 다른창 또 나오면 Configure 클릭

이제 이 페이지를 채워 주시면 됩니다.

Website URL : 여러분의 블로그 주소를 풀로 적어주세요.

다른건 안채워 주셔도 상관없습니다.
