Enumerations(열거형)

연관된 값의 그룹에 대해 공통 타입을 정의한 뒤 type-safe 하게 해당 값들을 사용 가능

  • 열거형 타입의 이름은 Pascal Case
  • 각각의 case는 Camel Case

타입을 연관 지을수가 있다.

값도 넣을 수 있다.

Raw Value

  • Strings, Characters, or any of the Integer or Floating-point number types

  • raw value 는 해당 Enumeration 내에서 반드시 고유한 값이어야 함.

rawvalue는 기본값이 0임

Collection

종류

Value Type (Swift Collection)

  • Array
  • Dictionary
  • Set

Object Type (Foundation Collection)

  • NSArray, NSMutableArray
  • NSDictionary,NSMutableDictionary
  • NSSet, NSMutableSet

Array

  • 값이 중복될 수 있으며 순서가 보장된다.
  • 순서의 시작은 0

선언방법

타입선언

method

.isEmpty : 값이 비어있는지 체크 -> Bool

.startIndex : 시작 인덱스 값 -> Int

.endIndex : 마지막 인덱스 값 -> Int

.contains(value) : value가 들어있는지 체크 -> Bool

.firstIndex(of: value) : 해당 value가 들어있는 첫번째 위치 -> Optional
